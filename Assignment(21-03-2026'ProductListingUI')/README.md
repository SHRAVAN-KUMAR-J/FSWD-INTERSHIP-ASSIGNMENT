# Product Listing UI - Mini E-commerce Frontend

## 📋 Assignment Overview
Build a mini e-commerce frontend with product cards and filters. This project demonstrates a modern, responsive product listing page with category filtering capabilities.

## 🚀 Features
- **Product Cards**: Display products with images, name, price, rating, availability status
- **Category Filtering**: Filter products by categories (Electronics, Clothing, Home, Accessories)
- **Responsive Design**: Fully responsive grid layout that adapts to all screen sizes
- **Interactive UI**: Hover effects, smooth transitions, and visual feedback
- **Cart Demo**: Basic cart functionality with item count indicator
- **Empty State**: Graceful handling when no products match filters

## 🛠️ Tech Stack
- **React 18** - UI library
- **CSS3** - Styling with modern features (Grid, Flexbox, animations)
- **Font Awesome 6** - Icon library
- **Google Fonts** - Inter font family
